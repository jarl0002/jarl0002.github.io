LIB2024- QA

  1. Qus: What are the three componets of Web?

    Ans: Ans for Qus 1: HTML, CSS, and JavaScript.

  2. Qus: Define version control and give one example.

    Ans: Ans for Qus 2: It's a system that records changes to a file/files over time so you can recall specific versions later. An example of version control would be GitHub.

  3. Qus: What tag is used to create the title of a web page? Give an example.

    Ans: Ans for Qus 3: <title> is the tag used to create the title of a webpage. An example would be <title> Learning How to Code </title>

  4. Qus: What tag is used to create a paragraph in html? Give an example.

    Ans: Ans for Qus 4: <p> is the tag used to create a paragraph in HTML. An example would be <p> There are many species of spiders in the world. </p>
